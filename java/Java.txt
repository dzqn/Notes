# Java Notlarım

* c sharp da "selead" class'lar java da "final" olarak tanımlanır.
Bu şekilde tanımlanan class field'leri miras yolu ile başka class'lara aktarılamaz.

* static metodlar c sharptan biraz farklı çalışıyor. static constructer tanımlaması yapılabiliyor.

* class içine class tanımlamaları (static) yapılabiliyor. (inner classlar)
Java da ana class static olamaz ama içindeki metodlar static olabilir. C sharp da class static olabilir.
